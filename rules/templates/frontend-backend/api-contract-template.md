## 9. 验收与门禁
1. 前端门禁：lint + typecheck + test 通过。
2. 服务端门禁：对应质量门禁通过。
3. 联调覆盖：成功路径 / 鉴权失败 / 参数错误 / 幂等重试。
4. 发布策略：先服务端兼容发布，再前端切换。
5. 回滚策略：服务端版本可回退 + 前端开关可回退。
