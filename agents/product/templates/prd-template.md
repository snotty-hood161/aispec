## 11. 衔接说明

### → Design Agent
以下内容需交由 Design Agent 处理：
- 第 4 节用户画像 → 设计目标用户的交互体验
- 第 5 节用户故事 → 设计用户操作流程和页面跳转
- 第 6 节功能清单 → 设计各功能模块的界面原型

### → Spec Agent
以下内容需交由 Spec Agent 处理：
- 第 6 节功能清单 → 技术选型与模块拆分
- 第 7 节非功能性需求 → 架构设计与约束定义
- 第 6.2 节业务规则 → 数据结构与 API 设计

### → QA Agent
以下内容需交由 QA Agent 处理：
- 第 5 节验收标准 → 测试用例设计
- 第 7 节非功能性需求 → 性能/安全测试策略
